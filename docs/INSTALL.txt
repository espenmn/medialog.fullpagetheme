medialog.fullpagetheme Installation
-----------------------------------

To install medialog.fullpagetheme using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``medialog.fullpagetheme`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        medialog.fullpagetheme
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
